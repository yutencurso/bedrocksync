**Description:**

This program is used to synchronize your Minecraft Bedrock mod project (regardless of which folder it's in) with the `development_behavior_packs` and `development_resource_packs` folders so that you can test the mods in real time.

**Commands:**
· *bedrocksync start* to start the sync with minecraft bedrock

· *bedrocksync stop* to stop the sync with minecraft bedrock

· *bedrocksync --version* show the version of the program

**Aviable in...**

only for windows
